1. Question: What are the three components of web?
    Answer: The 3 Components of web are HTML, CSS, and Javascript

  2. Question: Define version control and give one example.
    Answer: Version Control is a system that records changes to a file over time so that you can see the changes later. An example of this would be GitHub.

  3. Question: What tag is used to create the title of a web page?
    Answer: <title>Title</title>

  4. Question: What tag is used to create a paragraph in html?
    Answer: <p>This is a paragraph</p>

  5. Question:Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?
    Answer:The <p> tag is the odd one out because it is a Block element whereas the rest are inline elements.

  6. Question: What are the tags required to create a navigation list in HTML?
    Answer: The tags required to create a navigation list in HTML are an ordered or unordered list tag (<ol> or <ul>), a list item <li>, and finally a link tag <a>.
